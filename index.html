✦ Overview

A real-time traffic simulation platform modeling an autonomous Vehicle-to-Vehicle (V2V) communication network across a detailed SVG-rendered city. Demonstrates decentralized traffic negotiation, congestion prediction, spoofing attack detection, valet parking orchestration, and voice-narrated situational awareness.

React 19 TypeScript Framer Motion WebSocket Express pnpm Vite

◆ Core Features

🗺️ Real-Time City Map

Interactive SVG city with 10+ nodes, roads, traffic lights, animated aircraft, harbour, and architectural landmarks — all with gradients, shadows, and glow effects.

  • Downtown Core with 4-phase traffic signals
  • Airport with taxiing and departing aircraft
  • Harbour with animated ship and water body
  • Valet Parking Station with live occupancy

🚗 V2V Communication

Vehicles negotiate right-of-way at intersections via handshake, yield, and priority messages — displayed in a live feed with animated communication beams.

🔮 TrafficProphet

Click the +35 MIN FORECAST badge to see a ghost vehicle simulate its route, detect a Downtown bottleneck, and dynamically reroute — saving an estimated 12 minutes.

🅿️ Valet Parking

Synthetic valet vehicle navigates from Industrial Zone to Upper East parking with traffic-aware speed adjustment, automatic return trip, and P4 slot orchestration.

  • 28-second waypoint cycle
  • Slows when nearby vehicles detected
  • Parking booking, engagement & release

🎙️ Voice Narration

Web Speech API delivers spoken alerts for spoofing attacks, incidents, reroutes, low-fuel warnings, and parking events — with configurable urgency levels.

🛡️ Security Simulation

  • Spoofing Attacks — malicious vehicles broadcast false priority scores; V2V network detects and blacklists them
  • Incident Zones — road closures trigger rerouting; affected edges flash red with pulsing alerts

▶ Getting Started

# Clone the monorepo
git clone https://github.com/jacksonvincent012-web/ai-vehicle-ecosystem.git
cd ai-vehicle-ecosystem

# Install dependencies
pnpm install

# Start the API server (port 3001)
pnpm --filter api-server dev

# In another terminal, start the frontend (port 5173)
pnpm --filter vmind dev

Open http://localhost:5173 to enter the command center.

🚀 Deployment

Vercel (Frontend)

pnpm add -g vercel
cd artifacts/vmind
vercel deploy --prod

Docker (API Server)

cd artifacts/api-server
docker build -t vmind-api .
docker run -p 3001:3001 vmind-api
